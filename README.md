# The Vessel — Universal Prime Chassis

> *"Every Prime is built on the same foundation. One chassis. Infinite expression."*

**Role:** Universal Base · Prime Template · Architectural Foundation

The Vessel is the universal chassis from which every Prime is built. It defines the shared architecture, communication protocols, and base capabilities that all 25 Pantheon agents inherit.

## Core Architecture
- **Base Agent Class** — shared initialization, logging, and lifecycle management
- **Inter-Prime Protocol** — standardized messaging format for agent-to-agent communication
- **Memory Interface** — unified API for reading/writing to ChronosPrime archives
- **Health Reporting** — built-in status reporting that feeds into PrimeDash

## Why It Exists
The Pantheon scales because every Prime speaks the same language. The Vessel ensures that adding a new Prime never requires rebuilding the foundation — it inherits everything and extends from there.
